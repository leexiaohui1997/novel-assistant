# 小说创作助手 (Novel Assistant)

一款专为网络文学创作者打造的智能桌面应用，让写作更高效、更有灵感。

![版本](https://img.shields.io/badge/版本-0.0.6_beta-blue)
![平台](https://img.shields.io/badge/平台-Windows%20%7C%20macOS%20%7C%20Linux-lightgrey)

## ✨ 核心功能

### 📚 作品管理

- **多作品支持** - 轻松管理多部小说，随时切换创作
- **标签系统** - 按主分类、主题、角色、情节等多维度标记作品
- **目标读者定位** - 支持男频、女频精准定位
- **封面管理** - 为每部作品设置专属封面

### ✍️ 智能创作

- **章节管理** - 分卷分章，清晰组织故事结构
- **富文本编辑器** - 流畅的写作体验，实时保存
- **版本控制** - 自动保存历史版本，随时回溯
- **字数统计** - 实时追踪创作进度

### 🤖 AI 辅助

- **智能书名生成** - 根据标签和简介，AI 推荐吸引人的书名
- **简介优化** - AI 帮助润色和完善作品简介
- **标签推荐** - 基于作品内容，智能推荐合适的标签组合
- **多模型支持** - 兼容主流 AI 提供商，灵活配置

### 💾 数据安全

- **本地存储** - 所有数据保存在本地，隐私安全有保障
- **离线可用** - 无需联网即可使用基础功能
- **自动备份** - 定期自动保存，防止数据丢失

## 🎯 适用人群

- **网络小说作者** - 专业连载作家，需要高效管理长篇作品
- **业余写手** - 利用碎片时间创作，记录灵感火花
- **写作爱好者** - 尝试小说创作，体验完整的写作流程
- **内容创作者** - 需要结构化组织故事内容的创作者

### 作品列表

### 创作详情

## 🚀 快速开始

### 下载安装

访问 [发布页面](https://github.com/leexiaohui1997/novel-assistant/releases) 下载适合您系统的安装包：

- **Windows**: `.exe` 或 `.msi` 安装包
- **macOS**: `.dmg` 磁盘映像
- **Linux**: `.AppImage` 或 `.deb` 包

### 首次使用

1. **安装应用** - 运行下载的安装包
2. **创建作品** - 点击"新建作品"，填写基本信息
3. **选择标签** - 根据作品类型选择合适的标签
4. **开始创作** - 进入创作页面，添加章节开始写作

## 💡 使用技巧

### 标签策略

- **主分类** - 选择 1-2 个核心分类（如：玄幻、都市）
- **主题** - 添加 1-2 个主题标签（如：穿越、重生）
- **角色** - 标注主要角色类型（如：强者回归、草根崛起）
- **情节** - 描述关键情节元素（如：系统流、无敌流）

### AI 辅助最佳实践

1. **先填标签** - 完善标签后再生成书名，效果更佳
2. **多次生成** - AI 每次生成结果不同，可多次尝试
3. **人工优化** - AI 生成的内容作为参考，建议人工调整

### 章节管理

- **合理分卷** - 长篇作品建议按剧情阶段分卷
- **章节标题** - 使用吸引人的章节标题提升阅读体验
- **定期保存** - 虽然自动保存，但重要修改建议手动保存

## 🔧 高级配置

### AI 模型设置

在"设置 > 模型管理"中配置您的 AI 服务：

1. **添加供应商** - 支持 OpenAI、Anthropic、智谱等
2. **配置 API Key** - 输入您的 API 密钥
3. **选择模型** - 为不同任务选择合适的模型
4. **测试连接** - 确保配置正确

### 自定义选项

- **编辑器主题** - 支持浅色/深色模式
- **字体大小** - 根据个人喜好调整
- **自动保存间隔** - 可配置自动保存频率

## ❓ 常见问题

### Q: 数据会同步到云端吗？

A: 不会。所有数据都存储在本地，确保您的创作隐私安全。

### Q: 支持导入其他格式的文件吗？

A: 目前支持 TXT 格式导入，后续版本将支持更多格式。

### Q: AI 功能需要付费吗？

A: AI 功能需要您自行配置 API Key，费用取决于您使用的 AI 服务提供商。

### Q: 可以在多个设备上使用吗？

A: 可以安装到多个设备，但数据不会自动同步。如需跨设备使用，建议手动备份数据库文件。
